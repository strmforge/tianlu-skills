# Neutrality Review

Use this review before promoting content into runtime-neutral dao or laws.

## Literal Scan

Reject or move to an overlay when content contains:

- project names;
- collaborator aliases;
- private system names;
- machine-local paths;
- private repository assumptions;
- runtime-specific claims outside adapters.

## Scenario Fingerprint Scan

Reject, generalize, or move to an overlay when content assumes:

- one private workflow;
- user-specific wording;
- private collaboration slang;
- one runtime's tool surface;
- one organization's process.

## Stranger Test

A user doing an unrelated project with a different agent runtime should still understand runtime-neutral dao and laws without private context.
