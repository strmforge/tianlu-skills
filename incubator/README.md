# Incubator

The incubator is the entrance for new methods.

A method in incubation is not rejected. It is waiting for scope, evidence, and placement.

## Incubation Template

```text
Title:
Signal:
Source:
Proposed trigger:
Proposed mechanism:
Initial scope:
Known failure modes:
Verification needed:
Placement candidates:
```
